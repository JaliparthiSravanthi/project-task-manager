# TaskMaster - Project & Task Manager

## Overview

TaskMaster is a full-stack Project and Task Management application that helps users organize projects, manage tasks, track progress, and monitor deadlines efficiently.

The application provides OTP-based authentication, project management, task tracking, progress monitoring, and a clean user interface.

---

## Features

### Authentication

* OTP-based Login
* User Verification
* Session Management

### Project Management

* Create Projects
* View Projects
* Project Dashboard
* Project Progress Tracking

### Task Management

* Create Tasks
* Mark Tasks as Complete/Incomplete
* Delete Tasks
* Due Date Management

### Productivity Features

* Progress Tracking
* Active Task Counter
* Completed Task Counter
* Search Tasks
* Filter Tasks

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
npm run dev
```

4. Open in browser

```text
http://localhost:3000
```
